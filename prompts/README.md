# Prompt Library

Curated research prompts organized by category. Pipe into vibe-trading:

```bash
vibe-trading run -p "$(cat prompts/market-scan.txt)" --max-iter 12
```

## Categories

| Category | File | Description |
|----------|------|-------------|
| Market Scan | `market-scan.txt` | Daily market overview: top movers, volume, volatility |
| Strategy Test | `strategy-test.txt` | Quick SMA/EMA crossover backtest template |
| Alpha Discovery | `alpha-discovery.txt` | Factor zoo benchmark on your universe |
| Risk Check | `risk-check.txt` | Portfolio VaR + stress test |
| Journal Review | `journal-review.txt` | Upload broker export → Shadow Account analysis |
| Idea Generation | `idea-generation.txt` | Research Autopilot: hypothesis-driven discovery |
